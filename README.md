## 🛠️ 사용기술 및 라이브러리
- **Framework**: React Native
- **Network**: Axios, React-query
- **State management**: AsyncStorage, Recoil
- **Other**: React-navigation, jwt-decode

## 📱 실행방법
### 공통
- npm install
### ios
- cd ios && pod install && cd ..
- npm run ios

### android
- npm run android
